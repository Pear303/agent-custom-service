# 秋江独钓 · 七言绝句展示页

## 项目简介

一个展示原创七言绝句《秋江独钓》的优雅 HTML 静态页面。  
以中国古典水墨美学为基调，呈现诗意阅读体验。

## 诗歌内容

```
秋江独钓

一叶扁舟逐水流
半江瑟瑟半江秋
斜阳影里渔歌远
白鹭洲头暮色收
```

## 功能特性

- ✅ **原创七言绝句** — 四句七字，押韵工整，意境完整
- ✅ **优雅视觉设计** — 纸墨色调 + 渐变背景 + 水墨纹理
- ✅ **标题与作者署名** — 清晰展示诗歌标题和作者
- ✅ **响应式布局** — 手机 / 平板 / 桌面全适配
- ✅ **入场动画** — 诗句逐行淡入，优雅呈现
- ✅ **国风装饰元素** — 水墨渐变背景 + 印章点缀
- ✅ **零依赖** — 单 HTML 文件，双击即可打开

## 如何使用

### 直接打开

双击 `index.html` 即可在浏览器中查看。

### 部署到公网（可选）

可使用任意静态托管服务：

```bash
# 示例：使用 Vercel CLI
npx vercel --prod

# 或使用 GitHub Pages
# 将 index.html 推送到 gh-pages 分支即可
```
